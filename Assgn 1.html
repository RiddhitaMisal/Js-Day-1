JavaScript console.log() Methods

Console methods in javascript provides access to the browser's debugging console.

console.log() method Outputs a message to the console.

console.info() method Outputs an informational message to the console.

console.table() method Displays tabular data as a table.

console.error() method Outputs an error message to the console.

console.warn() method Outputs a warning message to the console.

console.assert() method Writes an error message to the console if the assertion is false.

console.time() method Starts a timer (can track how long an operation takes).

console.count() method Logs the number of times that this particular call to count() has been called.
